# SQL Data Warehouse Project

## Overview

This project demonstrates how to build a **SQL-based Data Warehouse** using structured datasets.
It focuses on transforming raw data into meaningful analytical tables that support reporting and business insights.

The project includes **data ingestion, transformation, and analytical queries** to simulate a real-world data warehouse workflow.


## Tech Stack

* SQL
* Relational Database (PostgreSQL / MySQL / SQL Server)
* Data Warehousing Concepts (Star Schema / Fact & Dimension Tables)


## Features

* Data warehouse schema design
* Data transformation using SQL
* Analytical queries for business insights
* Clean and structured SQL scripts


## Goal

The goal of this project is to practice **data warehousing fundamentals and SQL-based data modeling** used in real-world data engineering projects.
